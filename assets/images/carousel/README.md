# Carousel Images

Placez vos images de carousel ici pour personnaliser l'apparence de votre application.

## Noms de fichiers recommandés:

### Pour le carousel de la page Home:
- `active.png` - Image pour "Stay Active" (violet/pourpre)
- `nutrition.png` - Image pour "Eat Healthy" (rose/rouge)
- `sleep.png` - Image pour "Sleep Well" (bleu cyan)
- `progress.png` - Image pour "Track Progress" (vert/turquoise)

### Images utilisées dans chaque page:
- **HomeTab** (carousel): Utilise les 4 images ci-dessus
- **SleepTab**: `sleep.png` - Bannière en haut de page
- **MealsTab** (Nutrition): `nutrition.png` - Bannière en haut de page
- **LocationTab** (Activity): `active.png` - Bannière en haut de page
- **SocialTab**: `progress.png` - Bannière en haut de page

## Format recommandé:
- Format: **PNG** ou **JPG**
- Ratio: **16:9** (paysage)
- Dimensions: **800x450px** minimum
- Taille: **< 500KB** par image
- Style: Images lifestyle/santé/bien-être

## Suggestions d'images:
- **active.png**: Personne qui court, fait du sport, nature active
- **nutrition.png**: Fruits frais, légumes colorés, repas sain
- **sleep.png**: Lit confortable, chambre sereine, nuages/nuit
- **progress.png**: Graphiques de progression, objectifs atteints

## Configuration actuelle:
Les images sont **déjà configurées** dans le code. Il suffit de placer vos images dans ce dossier avec les noms exacts ci-dessus. Si une image n'est pas trouvée, un dégradé de couleur sera affiché à la place.

## Où trouver des images gratuites:
- Unsplash.com (photos libres de droits)
- Pexels.com (photos gratuites)
- Pixabay.com (images et illustrations)

Recherchez: "fitness", "healthy food", "sleep", "progress tracker"
